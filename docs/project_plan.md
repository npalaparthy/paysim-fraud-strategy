# Project Plan

## Dataset

PaySim Synthetic Financial Dataset for Fraud Detection

## Project Objective

Use SQL and Python to analyze the PaySim fraud dataset, identify fraud patterns, create detection rules, measure rule performance, tune thresholds, and prepare the foundation for feature engineering and beginner fraud-risk modelling.

## Fraud Questions

1. Which transaction types show higher fraud concentration?
2. Do high-value transfers show higher fraud risk?
3. Can a simple fraud rule identify suspicious transactions?
4. How many false positives does the rule create?
5. Which threshold provides the best balance between fraud detection and review volume?
